# Zeega Player

##Dependencies

* jquery

## tasks

### update submodules

    git submodule udpate --init --recursive

### commit changes to submodule

    cd [path of submodule]
    git commit -am 'message'
    git push origin master

### when you update an html template file in the layers

    bbb copy
    or
    bbb release
